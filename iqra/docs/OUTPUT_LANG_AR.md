# اختيار لغة المخرجات

الوضع الافتراضي لعرض القيم في إقرأ هو العربية. يمكنك التبديل إلى الإنجليزية عبر المتغير البيئي `IQRA_OUTPUT_LANG`.

- الافتراضي: العربية
- قيم الإنجليزية المقبولة: `en`, `english` (غير حسّاس لحالة الأحرف)

أمثلة (Windows PowerShell):

- مؤقتًا لأمر واحد:
  `$env:IQRA_OUTPUT_LANG = 'en'; cargo run -q -- examples/hello.iqra`
- تعيين للجلسة الحالية:
  `$env:IQRA_OUTPUT_LANG = 'english'`

ما الذي يتغير:

- المنطقي: `صحيح/خطأ` ↔ `true/false`
- لاشيء: `لاشيء` ↔ `nil`
- عرض الدالة: `<دالة>` / `<دالة مدمجة>` ↔ `<function>` / `<native>`

ملاحظات:

- يؤثر فقط على العرض (to_string والمطبوعات). لا يغيّر دلالات اللغة أو الكلمات المفتاحية المقبولة في الشيفرة.
- تظل العربية هي الافتراضية إذا كان المتغير غير معيّن أو معيّن لقيمة مختلفة عن `en`/`english`.
