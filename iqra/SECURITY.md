# Security Policy

## Supported Versions

This is a community project. We aim to keep the `main` branch and the latest release secure. Older releases may not receive security updates.

## Reporting a Vulnerability

- Please do not open a public issue for security vulnerabilities.
- Email the maintainer at <mailto:arjeic85@gmail.com> with details and a proof-of-concept if possible.
- We will acknowledge receipt within 72 hours and work on a fix.

## Disclosure

We prefer coordinated disclosure. Once a fix is available and released, we will credit reporters who wish to be acknowledged.
